# MathLove 🧮❤️

「ALL YOU NEED IS LOVE」——這是一個用 Swift 與 SwiftUI 打造的浪漫小專案，將生硬的數學算式化為愛意，在畫面上描繪出 **LOVE** 四個字母。

## 💡 創作靈感
本專案的靈感來自於一張刻著數學函數的項鍊照片。透過在直角坐標系上繪製特定的數學函數軌跡，完美拼湊出 L、O、V、E 的形狀，並將 O 調整為橢圓形，使整體視覺更加協調。

## 📐 背後的數學算式
這四個字母分別對應以下數學函數：

* **L** : y = 1/x (取第一象限的反比例函數)
* **O** : 橢圓方程式 (由原本的圓方程式 x^2 + y^2 = 9 調整長寬比例而來)
* **V** : y = |-2x| (絕對值函數的 V 型折線)
* **E** : x = -3|sin y| (正弦函數加上絕對值與反轉，取 -π 到 π 範圍)

## 🛠️ 技術細節
* 使用 **SwiftUI** 開發，程式碼簡潔易讀。
* 利用 `Path` 與 `GeometryReader` 打造自訂的繪圖視圖 (GraphView)。
* 包含灰色的 (x, y) 座標輔助線，重現紙上畫圖的真實感與精準度。

## 🚀 如何執行
1. 確認你的 Mac 已安裝 Xcode。
2. 將此專案 Clone 到本地：
   git clone https://github.com/Flyingcat10235/MathLove.git
3. 在 Xcode 中打開專案。
4. 在右側的 Preview (Canvas) 即可看見畫面，或按下 Cmd + R 在模擬器中執行。
